HOW TO ADD JUNIT5 TO INTELLIJ:
1. Top Bar File
2. File > Project Structure
3. This opens a new window, select "Modules" from the LEFT bar
4. Got to the "Dependencies" tab
5. Click the "+" symbol on the RIGHT bar
6. "+" > Library > From Maven...
7. Give this as address:
org.junit.jupiter:junit-jupiter-api:5.6.0
8. Add as module
9. Apply
10. Run the "TestCase" class file to make sure it works.
